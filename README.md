## 🚀 **Featured Projects**

### 🛸 **Edge-AI Modular Payload for Autonomous Infrastructure Inspection**
* **Overview:** An autonomous drone system built to navigate complex structural environments and detect real-time structural defects. Developed and pressure-tested under a rigorous 48-hour timeline, achieving a **Top-5 finish** at the BGI Hackathon 2026.
* **Tech Stack:** Webots (3D Digital Twin Simulation), Custom YOLOv11n, Supabase (Backend-as-a-Service), OpenCV, Raspberry Pi, ESP32-CAM.
* **Key Feature:** Engineered an autonomous waypoint flight stack and an onboard "Black Box" compute hub that flags structural anomalies locally without relying on cloud computation.

### 🌱 **Smart AI Drone for Leaf-Level Precision Pesticide Sprinkling**
* **Overview:** An AI-driven precision agriculture drone designed to reduce chemical waste by identifying crop illnesses and executing micro-targeted spraying.
* **Tech Stack:** Arduino Uno R3/R4, Raspberry Pi 5, ESP32-CAM, OpenCV, Tinkercad (3D Modeling & Circuit Design).
* **Key Feature:** Features a 1-axis robotic spray arm that triggers exclusively over infected leaves, cutting down pesticide expenses by up to 50%.

### 🌌 **Stellar Motion Analysis via Spectral Redshift**
* **Overview:** A scientific computing pipeline developed to analyze real stellar spectrum data, identify Hydrogen-alpha absorption lines, and calculate stellar velocities.
* **Tech Stack:** MATLAB, Vectorized Array Operations, Logical Indexing.
* **Key Feature:** Successfully processed multiple star datasets simultaneously to calculate precise redshift-based velocities.
